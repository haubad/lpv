Light Photo Viewer

Light Photo Viewer est une application légère et rapide pour visualiser des photos de votre ordinateur. Elle supporte les formats connus.


Fonctionnalités sont :

  • Charger et rendre la photo instantanément
  • Zoomer la photo à un point spécifique
  • Pivoter la photo
  • Supporter les formats BMP, GIF, ICO, JPEG, PNG, TIFF, WDP, HDP
  • Afficher les métadonnées Exif de la photo
  • Pivoter la photo automatiquement en utilisant les métadonnées Exif si elles existent
  • Mode panoramique
  • Redimensionner la photo


Téléchargement

L'application est gratuite Light Photo Viewer v1.70401 (January 04 2011, English version, 2,64 MB).

Système minimum requis

  • Windows 7, 8, 10 (recommandés), Windows Vista SP2
  • DirectX. Si un message d'erreur s'affiche comme "d2d1.dll and dwrite.dll not found", vous devez installer un des paquets suivants :
    • Update for Windows Vista (32 btis)
    • Update for Windows Vista (64 bits)
    • DirectX SDK

Pour visualiser des RAWs, vous devez avoir les codecs installés sur votre ordinateur :

  • Canon raw codec
  • Nikon raw codec - NEF codec
  • Nikon raw codec - NRW codec
